# 社長AI 壁打ちインタビューツール

AIを活用して、経営者の考えや価値観をQ&A形式で体系的に収集・蓄積するインタビューツールです。

🔗 **アプリURL**: https://bluebird4320.github.io/presidentAI/

---

## 概要

経営者へのインタビューをAIがサポートし、回答データをNotebookLMに取り込むことで、社員がいつでも経営者の考えを参照できる社内AIナレッジベースの構築を目的としています。

---

## 主な機能

- **AI質問自動生成** — Gemini APIがカテゴリに合わせた質問を自動生成
- **プリセット質問** — APIキー未設定でもデフォルト質問で利用可能
- **自動保存** — 回答はブラウザのlocalStorageにリアルタイム保存
- **カテゴリ管理** — カテゴリの追加・編集・削除が可能
- **データ出力** — MD / JSON / CSV形式、カテゴリ別エクスポート対応
- **音声入力** — Web Speech APIによる音声での回答入力（Chrome / Edge対応）
- **モバイル対応** — スマートフォンでも使用可能

---

## カテゴリ一覧

| カテゴリ | 内容 |
|---|---|
| 経営判断・事業方針 | 経営の意思決定・事業戦略に関する質問 |
| 人材・評価・採用 | 採用基準・評価・人材育成に関する質問 |
| 顧客対応・営業方針 | 顧客対応・営業スタンスに関する質問 |
| 社内文化・働き方 | 組織文化・労働環境に関する質問 |
| 会社の発展・将来ビジョン | 中長期の経営ビジョンに関する質問 |
| 給与・福利厚生・休暇 | 報酬・待遇に関する質問 |
| NG | やってはいけないことに関する質問 |
| 禁止事項 | 社内規定・禁止行為に関する質問 |

---

## 使い方

1. **カテゴリを選ぶ** — 左サイドバー（PC）または上部（スマホ）からカテゴリを選択
2. **質問に答える** — AIまたはプリセットの質問に回答を入力（テキスト or 音声入力）
3. **データを出力する** — 右上「📥 データ出力」からMD / JSON / CSV形式でダウンロード
4. **NotebookLMに取り込む** — ダウンロードしたMDファイルをNotebookLMにアップロード

## セットアップ

### 1. アプリにアクセス

以下のURLをブラウザで開いてください（推奨：Chrome / Edge）

https://bluebird4320.github.io/presidentAI/

### 2. Gemini APIキーを設定（任意）

1. [Google AI Studio](https://aistudio.google.com/app/apikey) でAPIキーを取得
2. アプリ右上の「⚙️ 設定」からAPIキーを登録
3. カテゴリ選択時にAIが質問を自動生成するようになる

> APIキー未設定でもプリセット質問で利用可能です。

---

## セキュリティ

| 項目 | 内容 |
|---|---|
| 回答データの保存先 | ブラウザ内（localStorage）のみ。外部サーバーへの送信なし |
| Gemini APIへの送信内容 | 質問生成のみ。回答内容は一切送信しない |
| APIキーの保管 | localStorageに保存。専用端末での使用を推奨 |
| URLアクセス | URLを知っている人のみアクセス可能 |
